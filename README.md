# Flask UI Template


Setup a virtualenv:

if not available
```
sudo apt install python3.8-venv
```

```
python3 -m venv .env
source .env/bin/activate

pip install -r requirements.txt
```

How to run?
```
python app.py
```

Template:
[Flask UI Template](https://github.com/rajasgs/flask-ui-template)
